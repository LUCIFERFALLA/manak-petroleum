# Manak Petroleum — Website

Static marketing website for **Manak Petroleum** (lubricating greases & engine oils, est. 1973),
manufactured at Harohalli, Bengaluru. Brands: **ZIGMA** and **EMTEX**.

## Stack
Plain, dependency-free **HTML + CSS + JS** (no build step). Fonts via Google Fonts;
images & video via Cloudinary CDN.

## Pages
- `index.html` — Home
- `about.html` — About / Heritage
- `products.html` — Product catalogue (ZIGMA + EMTEX)
- `product.html` — Sample product spec sheet
- `private-label.html` — Private label / toll blending
- `brands.html` — ZIGMA & EMTEX
- `industries.html` — Industries served
- `distributors.html` — Become a distributor
- `contact.html` — Contact

## Structure
- `css/main.css` — design system + all styles
- `js/main.js` — nav, reveals, counters, video play/pause, accordions

## Local preview
Open `index.html` in a browser, or run `npx serve .`
